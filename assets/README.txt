IMAGES NEEDED FOR THE WEBSITE
=============================

Drop the following files into this "assets" folder. Keep the exact file names
(the site looks for them by name). Until you add them, a striped placeholder
with the filename label is shown.

1. profile.jpg      - Your photo for the hero section.
                      Recommended: portrait, ~600 x 720 px (5:6 ratio).

2. project-1.jpg    - AI Resume Builder screenshot/cover.
3. project-2.jpg    - NLP Dashboard screenshot/cover.
4. project-3.jpg    - Fraud Detection System screenshot/cover.
5. project-4.jpg    - MLOps Platform screenshot/cover.
6. project-5.jpg    - Heart Disease Prediction screenshot/cover.
                      Recommended for projects: landscape, ~800 x 500 px (16:10).

7. Arnav-Jain-Resume.pdf  - Your resume PDF (for the "Download Resume" button).
                            Place it here with this exact name.

TIP: .png or .webp also work — just rename them to the names above (e.g.
profile.jpg can actually be a PNG renamed to profile.jpg, most browsers cope),
or update the file paths in index.html if you prefer different extensions.
